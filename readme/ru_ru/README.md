## О проекте

**Revoria** - глубоко переработанный лаунчер на базе экосистемы Modrinth / Theseus, ориентированный на реальные задачи игроков и сборщиков.

Проект поддерживается **sawiq** и публикуется здесь:

- Репозиторий: https://github.com/imsawiq/Revoria
- Релизы: https://github.com/imsawiq/Revoria/releases

---

## Почему Revoria

Revoria создан для тех, кому важны:

- новый и чистый дизайн лаунчера;
- больше контроля над системой (память, профили, инструменты);
- поддержка CurseForge наряду с Modrinth;
- полноценный русский язык (в будущем - новые локализации);
- встроенный AI-анализ crash-логов;
- поддержка офлайн-аутентификации;
- отсутствие рекламы.
- и ОЧЕНЬ много другого нового

---

## Ключевые возможности

- Полностью обновлённый интерфейс.
- Новый менеджер памяти.
- Поддержка CurseForge.
- Полная поддержка русского языка (и расширение локализаций в будущем).
- AI Log Checker (анализ crash-логов через Ollama API).
- Новая вкладка со скриншотами в сборке/инстансе.
- OFFLINE AUTH.
- NO ADS.
- Discord Rich Presence.

## Установка

1. Откройте релизы: https://github.com/imsawiq/Revoria/releases
2. Скачайте файл под вашу ОС.
3. Установите и запустите.

### Форматы

- `.msi` / `.exe` - Windows
- `.dmg` - macOS
- `.deb` - Linux

---

## Быстрый старт

1. Запустите Revoria.
2. Войдите через Microsoft аккаунт **или** используйте офлайн-режим.
3. Создайте/выберите сборку.
4. Устанавливайте моды, ресурспаки и шейдеры из поддерживаемых источников.
5. Играйте.

---

## Ollama API Key (AI Crash Checker)

Чтобы использовать AI-анализ crash-логов:

1. Авторизуйтесь на https://ollama.com/
2. Перейдите в раздел ключей: https://ollama.com/settings/keys
3. Нажмите **Add API Key**
4. Скопируйте ключ и вставьте его в Revoria -> Crash Log Checker -> API key
